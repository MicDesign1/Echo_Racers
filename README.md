# Animalian: Echo Racers

Creature-powered pseudo-3D racing game, sequel to Animalian Manor. React + Vite,
canvas 2D sprite-scaling road rendering (OutRun-style), localStorage saves.
Deployed to Cloudflare Pages.

See [CLAUDE.md](./CLAUDE.md) for the full project brief.

## Dev

```
npm install
npm run dev
```
